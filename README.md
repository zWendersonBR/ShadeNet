ShadeNet Cipher: Secure TCP/IP Communication (C# / .NET)ShadeNet Cipher is a command-line application built in C# (.NET) for demonstrating and studying asynchronous TCP/IP sockets. It features a Server (ShadeNetServer) that can handle multiple connections and a Client (ShadeNetClient) for connecting and engaging in basic console-based chat.üöÄ PrerequisitesTo compile and run this application, you must have the .NET SDK installed.
